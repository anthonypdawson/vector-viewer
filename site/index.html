Vector Inspector

A comprehensive desktop application for visualizing, querying, and managing vector database data.

Vector Inspector provides an intuitive GUI for exploring vector embeddings, metadata, and performing similarity searches across multiple vector database providers.

Quick Install:
macOS / Linux:
curl -fsSL https://vector-inspector.divinedevops.com/install.sh | bash
Windows:
powershell -c "iwr https://vector-inspector.divinedevops.com/install.ps1 -UseBasicParsing | iex"
Or install with pip:
pip install vector-inspector
vector-inspector
Supported providers: Chroma, Qdrant, Postgres (pgvector), Pinecone.
More backends coming as the ecosystem evolves.
  • Browse and search vector data
  • Visualize embeddings and metadata
  • Inspect, debug, and compare providers
